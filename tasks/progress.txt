## Codebase Patterns
- Nested config defaults can live directly in the Zod schema in `packages/opencode/src/config/config.ts`; cover them with `Config.get()` integration tests in `packages/opencode/test/config/config.test.ts` so parsing and merge behavior are validated together.
- Keep Hindsight bank identity and resolve rules in dedicated helpers under `packages/opencode/src/memory/hindsight/`; only `mem:` ids should resolve directly to authoritative memories, while `sess:` and `obs:` remain indirect ranking evidence and cross-worktree hashes must be dropped.
- Workspace-scoped companion services fit `Instance.state(..., dispose)` well: cache one lazy boot promise per worktree, bind the embedded daemon to `127.0.0.1`, and treat startup or health timeouts as degraded fallback instead of throwing into memory flows.
- When Hindsight needs shared timeout or fallback behavior across retain, recall, and inspect paths, wrap the SDK once in `packages/opencode/src/memory/hindsight/client.ts` and keep downstream memory flows on non-fatal `undefined` results instead of scattering per-call error handling.
- Authoritative-memory retain helpers should build `document_id`, metadata, and tags through `MemoryHindsightMap`, force `update_mode: "replace"`, and return structured non-fatal results so local writes never depend on companion success.
- Hindsight recall wrappers should tolerate SDK hit-shape drift (`results` vs `items`, `score` vs relevance-score aliases), preserve source rank on resolved candidates, and record stale or indirect drops separately so downstream recall only sees authoritative local memories.
- Engine recall should enrich `memory-recall` candidate JSON with optional Hindsight `rank` and `score` hints, but fall back to the original full pool whenever semantic recall yields no authoritative candidates.
- Authority-boundary regressions should assert local `personal.json`, injector payloads, hit counters, and decay behavior still move only through local memory records even when tests enable Hindsight; in typed fixtures, pass the full nested Hindsight defaults object into `tmpdir({ config })`.
