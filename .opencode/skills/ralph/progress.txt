## Codebase Patterns
- Swarm admin read models live in `packages/opencode/src/session/swarm-admin.ts`; keep browser pages on aggregated fields instead of stitching board JSON client-side.
- Safe delete is metadata-only via `Swarm.Info.time.deleted`; stop records `Swarm.Info.time.stopped` while board files stay intact.
- Swarm admin pages use polling plus Swarm SSE for refresh, and inline confirmation panels instead of browser-native dialogs.
- `./packages/sdk/js/script/build.ts` cleans `packages/sdk/js/src/v2/gen`, so do not run it in parallel with `packages/app` builds that resolve the workspace SDK source.
